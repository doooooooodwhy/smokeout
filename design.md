# Smokeout Game - Visual Design Style

## Design Philosophy
**"Competitive Fire"** - A bold, high-contrast design that captures the intensity of verbal combat while maintaining excellent readability for fast-paced text-based gameplay.

## Color Palette
- **Primary**: Deep Charcoal (#1a1a1a) - Main background, creates intensity
- **Secondary**: Fiery Orange (#ff6b35) - Accent color for active elements, timers, and power-ups
- **Tertiary**: Electric Blue (#00d4ff) - Player one color, cool counterpoint to orange
- **Quaternary**: Lime Green (#39ff14) - Player two color, vibrant and energetic
- **Neutral Light**: Soft White (#f5f5f5) - Primary text and UI elements
- **Neutral Dark**: Medium Gray (#666666) - Secondary text and inactive elements
- **Warning**: Bright Red (#ff073a) - Timer warnings and critical alerts
- **Success**: Bright Green (#00ff88) - Victory notifications and positive feedback

## Typography
- **Display Font**: "Orbitron" - Futuristic, competitive feel for headings and game titles
- **Body Font**: "Roboto Mono" - Monospace ensures perfect alignment for chat/roast displays
- **UI Font**: "Inter" - Clean, readable for buttons and interface elements
- **Accent Font**: "Russo One" - Bold, impactful for power-up names and special effects

## Visual Language
- **Geometric Shapes**: Sharp angles, hexagonal frames, and angular dividers
- **Grid System**: Clean, structured layouts with clear separation between teams
- **Contrast**: High contrast between light text and dark backgrounds
- **Motion**: Smooth transitions and animations for game state changes
- **Hierarchy**: Clear visual hierarchy with size, color, and spacing

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth animations for UI transitions and power-up activations
- **Typed.js**: Typewriter effects for dramatic roast reveals
- **Splitting.js**: Text animation effects for headings and announcements
- **ECharts.js**: Clean data visualization for leaderboards and statistics
- **p5.js**: Dynamic background effects and particle systems

### Animation Effects
- **Text Reveal**: Typewriter animation for roast messages appearing
- **Timer Pulse**: Pulsing animation for countdown timers
- **Power-Up Glow**: Glowing border effects when power-ups are activated
- **Score Counter**: Smooth number counting animations for roastbucks
- **Victory Flash**: Screen flash effects when rounds are won

### Header/Hero Effects
- **Dynamic Background**: Subtle particle system with floating geometric shapes
- **Gradient Overlay**: Animated gradient that shifts between orange and blue
- **Glitch Text**: Occasional glitch effect on main title for edginess
- **Competitive Frame**: Angular border frame around main game area

### Interactive Elements
- **Button Hover**: 3D lift effect with orange glow
- **Input Focus**: Electric blue border glow when typing roasts
- **Power-Up Cards**: Flip animation revealing power-up details
- **Leaderboard**: Smooth scroll and highlight animations

### Layout Structure
- **Split Battlefield**: Clean 50/50 split for team roasts
- **Central Hub**: Game controls, timers, and status in center
- **Sidebar Panels**: Power-ups, stats, and player info on sides
- **Bottom Bar**: Roast input and submission area
- **Top Bar**: Game mode, round info, and navigation

### Responsive Design
- **Desktop**: Full split-screen battle layout
- **Tablet**: Stacked layout with collapsible sidebars
- **Mobile**: Single-column layout with swipe navigation

### Accessibility Features
- **High Contrast**: 4.5:1 minimum contrast ratio for all text
- **Large Touch Targets**: Minimum 44px for all interactive elements
- **Clear Focus States**: Visible keyboard navigation
- **Screen Reader Support**: Proper ARIA labels and semantic HTML

## Game-Specific Visual Elements

### Battle Arena
- **Team Colors**: Blue vs Green with clear visual separation
- **Message Bubbles**: Angled, comic-book style speech bubbles for roasts
- **Turn Indicator**: Pulsing arrow or highlight showing current player
- **Timer Ring**: Circular countdown timer with color progression

### Power-Up Shop
- **Card Layout**: Grid of power-up cards with cost and description
- **Currency Display**: Prominent roastbucks counter with coin icon
- **Purchase Animation**: Flash and number decrement when buying

### Leaderboard
- **Ranking Display**: Numbered list with player names and stats
- **Category Tabs**: Switch between roasts, roastbucks, and time played
- **Achievement Badges**: Small icons for milestones and accomplishments

### Judge System
- **Vote Display**: Clear yes/no indicators for judge decisions
- **Intermission Timer**: 15-second countdown between rounds
- **Bracket Visualization**: Tournament bracket showing progression

This design creates an intense, competitive atmosphere perfect for roast battles while ensuring all text remains highly readable during fast-paced gameplay.
